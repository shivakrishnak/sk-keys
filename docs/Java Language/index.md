# Java Language
## Java Language Features & Internals
This section covers Java language-level features: generics, type system, reflection,
records, sealed classes, and modern Java enhancements.
### Topics Covered
- **Type System** — Generics, Type Erasure, Bounded Wildcards, Covariance
- **Reflection & Metadata** — Reflection API, Annotation Processing
- **Serialization** — Serialization, SerialVersionUID
- **Modern Java** — Records (Java 16+), Sealed Classes (Java 17+), Pattern Matching (Java 21+), invokedynamic
