# Identity & Access Management

Identity lifecycle, provisioning and deprovisioning, SCIM, directory
services (LDAP, Active Directory), SSO/federation (SAML), cloud IAM
(AWS, GCP, Azure), privileged access management, identity governance,
zero trust identity architecture, enterprise IAM patterns, and
specification convergence - from first principles to enterprise-grade
identity platform design.

> **Scope note:** Authentication mechanisms (password hashing, MFA,
> FIDO2, mTLS) live in [Authentication](../ATH-authentication/).
> Authorization models (RBAC internals, OPA, Zanzibar) live in
> [Authorization](../ATZ-authorization/). OAuth 2.0 flows live in
> [OAuth 2.0 & OpenID Connect](../OAU-oauth/).
